Investigate real primary-source evidence for agent-to-agent interoperability (Google A2A protocol, Agent Client Protocol / ACP), multiplayer/shared agent sessions, deterministic trace/replay and eval harnesses, budget/cost governance, and durable memory/knowledge layers. Write evidence-only report to `research/future-directions-source.md` under active Meridian work directory, with specific URLs and repo paths.
